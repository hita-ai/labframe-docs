# Quickstart

This guide walks you through creating an agent, sending a message, and opening a deliverable.

## Step 1: Sign in

Go to [labframe.ai](https://labframe.ai) and sign in with your university email. LabFrame uses magic-link authentication — check your inbox for the sign-in link.

## Step 2: Create an agent

Click **New Agent** in the sidebar. Give your agent a name and optionally customize its icon and system prompt. The agent starts with a fresh cloud workspace.

## Step 3: Send a message

Type a request in the chat input. For example:

> Build me a landing page for a student portfolio site.

Your agent will:
- Think through the approach
- Write HTML, CSS, and JavaScript files into the workspace
- Package the result as a `.webapp` artifact
- Present the finished deliverable in chat

## Step 4: Open your deliverable

Click the `.webapp` artifact link in chat. LabFrame deploys the static site and opens it in a new tab at a URL like `https://portfolio.labframe.ai/`.

You can also find the artifact in the **Workspace** tab and single-click it to see metadata, or double-click to open it directly.

## Step 5: Iterate

Send follow-up messages to refine the result. Your agent remembers the full conversation and has access to all the files it already created.
