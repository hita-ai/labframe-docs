# Collaboration

LabFrame supports real-time collaboration on agent conversations. You can invite others to work with the same agent, see the same workspace, and contribute to the same conversation.

## Sharing a conversation

To share a conversation (project):

1. Open the conversation you want to share
2. Click the **Share** button in the conversation header
3. Enter the collaborator's email address
4. Choose their access role

## Access roles

| Role | Can send messages | Can view workspace | Can manage sharing |
|------|:-:|:-:|:-:|
| **Owner** | Yes | Yes | Yes |
| **Editor** | Yes | Yes | No |
| **Viewer** | No | Yes | No |

## Cross-organization collaboration

LabFrame supports inviting collaborators from different organizations. When you share a conversation with someone from another org:

- They receive an email invitation with a link
- Clicking the link accepts the invite and opens the shared conversation
- Their messages appear with their name in the conversation

## Real-time updates

When multiple people are viewing the same conversation:

- New messages appear in real time for all participants
- Agent responses stream live to everyone
- Workspace file changes are visible after refresh

## Agent-level sharing

You can also share an entire agent (not just a single conversation). Agent-level collaborators can:

- See all conversations they have access to
- Create new projects under the shared agent
- Access the shared workspace
