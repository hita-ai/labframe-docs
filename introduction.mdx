title: What is LabFrame?
description: Persistent AI agents with dedicated workspaces, tools, and memory for research teams.
---

LabFrame gives you persistent AI agents that do real work. Each agent has a dedicated cloud workspace where it can write code, run commands, search the web, generate documents, and build websites — then hand you the finished deliverable.

It is not a chatbot. It is an AI teammate with its own computer.

## Who is it for?

LabFrame is built for R&D teams, university labs, and national labs. Researchers, students, and instructors use it to accelerate work that used to take weeks — literature reviews, data analysis, course material, prototypes, and technical writing.

## How a typical session looks

1. You create an agent and give it a name.
2. You send a message: *"Build a literature review on battery degradation models."*
3. The agent searches the web, reads papers, writes a LaTeX document, compiles it to PDF, and presents you with the file.
4. You click the PDF link in chat to preview it, or open the workspace to download it.
5. You send a follow-up: *"Add a comparison table and regenerate."* The agent iterates on the same files.

Everything the agent produces lives in its workspace. Nothing disappears when the conversation ends.
