# Welcome to LabFrame

LabFrame gives every user a personal AI agent backed by a full cloud development environment. Your agent can write code, generate documents, build websites, analyze data, and produce real deliverables — not just chat responses.

## What makes LabFrame different

<CardGroup cols={2}>
  <Card title="Real workspaces" icon="folder-open">
    Every agent has a persistent cloud workspace with files, tools, and a full Linux environment. Your agent writes real code into real files.
  </Card>
  <Card title="Deployable artifacts" icon="rocket">
    Agents produce deliverables you can use — PDFs, slides, datasets, and static websites deployed to their own URL.
  </Card>
  <Card title="Collaborative sessions" icon="users">
    Share a conversation with collaborators. Everyone sees the same agent, the same workspace, and the same results.
  </Card>
  <Card title="Built for education" icon="graduation-cap">
    Designed for university courses, research labs, and student projects. Instructors can provision agents for entire classes.
  </Card>
</CardGroup>

## How it works

1. **Create an agent** — each agent gets its own name, icon, system prompt, and cloud workspace.
2. **Chat with your agent** — ask it to build something. It writes code, runs commands, and produces files in its workspace.
3. **Open your deliverables** — click a generated `.webapp` to open a deployed website, download a PDF, or browse workspace files.
4. **Collaborate** — invite others to a shared conversation. They see the same workspace and can send their own messages.
