Skills are bundles of instructions and reference material that you can create once and make available to all your agents. Think of them as specialized knowledge packs.

## What is a skill?

A skill has three parts:

| Part | Description |
|------|-------------|
| **Name** | A short label (e.g., "Company Style Guide") |
| **When to use** | A description of when the agent should activate this skill |
| **Instructions** | Detailed guidance the agent follows when the skill is relevant |

You can also attach files to a skill — data sources, templates, or reference documents that the agent can read when the skill is active.

## Creating a skill

1. Click **Skills** in the navigation
2. Click **+ Create Skill**
3. Fill in the name, when-to-use description, and instructions
4. Optionally upload reference files
5. Click **Save**

The skill is now available to all your agents.

## How agents use skills

When you send a message, the agent checks whether any installed skill matches the task. If so, the agent follows that skill's instructions alongside its own system prompt.

For example, if you have a "LaTeX Paper" skill with instructions on your lab's preferred formatting, the agent will use those conventions whenever you ask it to write a paper.

## Built-in skills

LabFrame ships with built-in skills for common workflows:

- **Web Search** — enables the agent to search the web for current information
- **Hosted Web Apps** — guides the agent to package websites as `.webapp` deliverables
- **HTML Slides** — helps the agent produce presentation slide decks
- **PPTX Slides** — for generating native PowerPoint files
- **Research Project** — structured long-form research with evidence gathering

Built-in skills appear alongside your custom skills in the Skills Library.

## Editing and deleting

Hover over any custom skill in the Skills Library to see the edit (pencil) and delete (trash) icons. Built-in skills cannot be edited or deleted.
